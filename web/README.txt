프로젝트 설명: 사람인 채용 공고 스크래퍼
원래 수업 시간에 했던 인크루트 크롤링 과제를 바탕으로 만들었는데, 제가 자주 사용하는 취업 사이트인 '사람인' 데이터를 가져오도록 수정해봤습니다. 단순히 사이트만 바꾼 게 아니라 공부하면서 몇 가지 기능을 더 추가했습니다.

1. 보안 접근 권한 해결 (saramin.py)
인크루트와 달리 사람인은 프로그램의 자동화된 접근을 제한하고 있어, 초기 구현 시 데이터를 불러오지 못하는 문제가 있었습니다. 이를 해결하기 위해 headers에 User-Agent 정보를 추가하여, 브라우저를 통한 정상적인 접근임을 증명하도록 코드를 수정하였습니다.

2. 뒤로가기 버튼
검색 결과 페이지에서 다시 메인 화면으로 돌아가 새로운 키워드를 검색할 수 있도록 상단에 '다시 검색하기' 버튼을 배치하였습니다. 클릭 한 번으로 메인 화면으로 갈 수 있게 동선을 짰습니다.

3. 대구 지역 강조 표시
제가 대구 지역 공고에 관심이 많아서, 결과 리스트 중에 지역이 '대구'인 것들은 회사 이름이랑 지역명이 빨간색/굵은 글씨로 나오게 만들었습니다.

이 과정에서 '부산 해운대구'가 함께 강조되는 예외 상황을 발견했습니다. 이를 방지하기 위해 조건문에 공백을 포함한 "대구 " 형식을 사용하여, 오직 대구 지역 공고의 회사명과 지역명만 강조하여 출력되도록 정밀하게 수정하였습니다.
